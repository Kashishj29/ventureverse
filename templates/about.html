About VentureVerse

AI-powered startup success prediction using machine learning

ğŸ¯ Project Overview

VentureVerse is a machine learning-based web application that predicts the success potential of startup ideas using pre-launch indicators derived from publicly available data. The system analyses factors such as funding stage, team composition, industry type, and technology stack to generate a success probability score.

ğŸ“Š ML Models Evaluated

Three machine learning algorithms were trained and compared:

  • Logistic Regression - Baseline interpretable model
  • Random Forest - Ensemble method with feature importance
  • Gradient Boosting (XGBoost) - Advanced ensemble for best accuracy

The best performing model is automatically selected based on F1 score evaluation.

ğŸ“ˆ Dataset

The model is trained on a dataset of 5,000+ global startups with features including: funding amounts, employee counts, valuations, industry sectors, geographic regions, funding stages, and outcomes (IPO/Acquired status).
